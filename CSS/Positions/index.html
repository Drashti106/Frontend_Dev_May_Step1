Positions in CSS

  1. Static:the original or default position of an element
  2. Relative: Element is positioned relative to it's original/normal position. Same as static positioning, just that here we can move the element by adding offset values. Element stays in the normal document flow
  3. Absolute: 1) Element is positioned relative to its parent 2) if parent is not present then element is positioned relative to the main document 3) element is removed from the normal document flow, no space is created for the element.
  4. Fixed:element is always positioned ralative to viewport element alwyas statys in the same position even on the scroling element is removed from the normal document flow
  5. Sticky:element is positioned according to the noemal document flow till the element hits the top, it has normal position, after it thouches the top , it's positions becomes fixed
